# CML-080 — Curriculum and Didattica Completion Selection Audit

## Stato

Audit di completamento delle aree centrali: Curriculum (4 sub-tab) e Didattica (2 prototipi).

## Risultato analisi

| Area                    | Stato               | Gap                                                                     |
| ----------------------- | ------------------- | ----------------------------------------------------------------------- |
| Curriculum — Consulta   | Funzionante         | Mancano indicazioni esplicite sul percorso consulta→revisione→decisione |
| Curriculum — Revisione  | Funzionante         | Flusso `.cml` completo                                                  |
| Curriculum — Definitivo | Funzionante         | Export completo                                                         |
| Curriculum — Fonti      | Completo            | 4 categorie: istituzionali, PTOF/RAV, normativa, UE                     |
| Didattica — Evidenze    | Prototipo read-only | Da rendere operativo, mancano collegamenti ad attività/valutazione      |
| Didattica — UDA         | Prototipo read-only | Singolo modello, manca guida all'uso                                    |
| Didattica — Attività    | Non implementato    | "In preparazione"                                                       |
| Didattica — Materiali   | Non implementato    | "In preparazione"                                                       |

## Opzioni

| Opzione                   | Descrizione                                         | Rischio |
| ------------------------- | --------------------------------------------------- | ------- |
| A — Curriculum prima      | Allineato con flusso `.cml` e mandato istituzionale | Basso   |
| B — Didattica prima       | Utile per docenti ma perimetro ampio                | Medio   |
| C — Entrambe in una slice | Troppi cambiamenti insieme                          | Alto    |
| D → A                     | Audit poi Curriculum poi Didattica (raccomandato)   | Basso   |

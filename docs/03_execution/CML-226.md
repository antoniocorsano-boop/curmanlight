## Modifiche runtime effettuate

1. **CSS**: 10 classi `.tecnologia-export-*` → `.disc-export-*` (incluse print/responsive)
2. **HTML**: id e class dell'export panel e preview textarea rinominati
3. **JS**: `generateTecnologiaBozza()` → `generateDiscBozza()` usa `selDisc`
4. **JS**: `copyTecnologiaMarkdown()` → `copyDiscMarkdown()` usa `selDisc`
5. **JS**: `downloadTecnologiaMarkdown()` → `downloadDiscMarkdown()` usa `selDisc`
6. **JS**: Export panel sempre visibile (rimossa condizione `selDisc === 'Tecnologia'`)
7. **JS**: Fallback title Markdown generico ("Markdown curricolo")
8. **Testi**: Home e didattica generalizzati
9. **Default**: `selDisc` e `DEFAULT_PROFILE` da `DISCIPLINE[0]`

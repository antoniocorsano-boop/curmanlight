## Validazione dipartimentale

`renderDepartmentImport()` contiene:

```js
html+=grouped[discipline][status].map((proposal,idx)=>{
  ...
  var decision=decisionIdx>=0?departmentImportState.proposals[decisionIdx].decision:null;
  return '<article ... data-idx="'+decisionIdx+'" ... onchange="setDepartmentDecision('+decisionIdx+',this.value)" ...';
});
```

`decisionIdx` non e' definito prima dell'uso. Con mock valido di import `.cml`, `renderDepartmentImport()` genera:

```text
ReferenceError: decisionIdx is not defined
```

Classificazione: **P1 - import/validazione dipartimentale bloccante quando esistono proposte importate**.

Proposta per slice successivo: calcolare l'indice assoluto prima dell'uso, per esempio `const decisionIdx = departmentImportState.proposals.indexOf(proposal);`, o una mappa stabile id -> indice.

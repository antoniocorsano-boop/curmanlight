## Problema originario

Lo scaffold React in `curman-react/` non compilava per tre ragioni:

1. **Tipi mancanti**: `src/types/curriculum.ts` non esisteva ‚Äî 6+ file importavano `@/types/curriculum` (UnitaApprendimento, CurriculumDisciplina, DisciplinaSlug, DISCIPLINE_SLUGS, DISCIPLINE_LABELS)
2. **Libreria mancante**: `src/lib/curriculum.ts` non esisteva ‚Äî `useCurriculum.ts` importava `filterByOrdine`, `filterByNucleo`, `mergeGapLayer`
3. **Dati assenti**: `import.meta.glob('../data/curriculum/*.json')` puntava a una directory vuota
4. **Lockfile fittizio**: `package-lock.json` conteneva solo `see_file` ‚Äî `npm ci` impossible

## Decisioni chiave

- **Fonte canonica**: `content/curriculum/*.normalized.json` ‚Äî unica fonte dei dati curricolari
- **Dati generati non versionati**: `src/data/curriculum/*.json` sono artefatti derivati, esclusi da git tramite `.gitignore`
- **pages.yml invariato**: `_published_snapshot/netlify-current/index.html` rimane l'unico target di pubblicazione su GitHub Pages
- **react-ci.yml come sola verifica**: nessun deploy, nessun `pages:write`, nessuna pubblicazione
- **Gap layer assente**: Revisione e Esportazioni mostrano banner informativo, non liste vuote

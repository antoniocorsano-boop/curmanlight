## Selezione finale

### Scelta: Opzione D — Pacchetto esempi dimostrativi

**Prossimo step:** CML-031 — Creazione file `.cml` di esempio (proposta docente, esito dipartimento) e breve guida alla simulazione.

**Motivazione:**

1. **Valore reale alto:** I destinatari del pacchetto handoff (DS, docenti, coordinatori, referente) non hanno un modo per provare lo strumento senza usare dati veri. File `.cml` di esempio permettono di simulare l'intero flusso in 5 minuti, durante una riunione di formazione o una presentazione.
2. **Rischio nullo:** Docs-only. File `.cml` di esempio in una cartella `docs/04_user/esempi/`. Nessuna modifica runtime, nessun deploy.
3. **Completa il pacchetto handoff:** I documenti di presentazione spiegano "cosa fare"; gli esempi mostrano "come si fa".
4. **Non brucia opzioni future:** Le opzioni A, B, C, F restano disponibili per i prossimi cicli.
5. **Nessuna modifica a schema `.cml`:** I file esempio usano il formato esistente.

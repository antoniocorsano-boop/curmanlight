# CML-068 — Didattica — First Read-Only Prototype (Valutazione/Evidenze)

**Stato:** ✅ Completed  
**Data:** 2026-06-23  
**Commit:** `feat: add read-only Didattica evidence prototype`  

## Summary

Implement first read-only prototype of the Didattica area in CurManLight, corresponding to the Valutazione/Evidenze module (Option D selected in CML-067). The prototype displays the 13 Tecnologia unità di apprendimento with their evidenze osservabili and criteri di valutazione in an expandable card layout with ordine filter (Tutti/Infanzia/Primaria/Secondaria), read-only, no persistence, no Curriculum area modifications.

## Changes Applied

### index.html (sole runtime file)

1. **CSS section** — Added `.didattica-evidence-*` styles (lines ~600s): evidence-section, unit card, filter bar, stat bar, note, responsive layout. All within existing `/* CURRICOLO DI ISTITUTO */` area.

2. **HTML tab section** — Inserted `#tab-didattica` (after `#tab-home`): contains prototype badge, stats row, ordine filter buttons (Tutti/Infanzia/Primaria/Secondaria), and `#didattica-evidence-list` container.

3. **Tabbar button** — Added "Didattica" button between Home and Lavoro.

4. **Home card Didattica** — Three links: first ("Valuta evidenze") active; second ("Verifica competenze") and third ("Programmazione didattica") set to `#` with `.dimmed` class; fourth ("Documenti e materiali") also `#` + `.dimmed`.

5. **Mobile menu** — Added `🧑‍🏫 Didattica — Valutazione ed evidenze` entry.

6. **setTab()** — Extended for `didattica` tab: visibility toggle, `renderDidattica()` call, cruscotto/breadcrumb hide, tabbar map, breadcrumb label, mobile bottom bar map (`mbb-home`).

7. **Data + render** — Embedded `TECNOLOGIA_NORM_DATA` (13 unitàApprendimento with evidenze, criteriValutazione fields) as a JS variable; `renderDidattica()` function building expandable unit cards with obiettivi, conoscenze, abilità, evidenze, criteriValutazione sections; `setDidatticaFilter()` for ordine filtering.

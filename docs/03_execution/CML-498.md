# CML-498 — Import multiplo delle proposte `.cml`

## Obiettivo

Consentire al ruolo Dipartimento di selezionare e verificare più file di proposta docente `.cml` in un solo passaggio, mantenendo il trattamento completamente locale.

## Implementazione

- aggiunta una superficie dipartimentale nella vista `Esportazioni e passaggi di lavoro`;
- selezione multipla di file `.cml` tramite file picker nativo;
- parsing con il contratto CurManLight v1.0 già esistente;
- classificazione per file: valido, duplicato, non valido;
- rilevamento dei duplicati nel gruppo selezionato tramite fingerprint del contenuto;
- riepilogo di file, proposte pronte, duplicati e file non validi;
- metadati visibili: disciplina, ordine, profilo e numero di proposte;
- esclusione automatica dei duplicati e dei file non validi dal conteggio delle proposte pronte;
- nessun upload, backend, telemetria o modifica dei file originali.

## Confini

Questa slice prepara e valida il gruppo di ingresso. La persistenza della coda dipartimentale e le decisioni sulle singole proposte restano assegnate a CML-499 e CML-500.

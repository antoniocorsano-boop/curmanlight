**2. `getMappaForDiscipline(discKey)`**
```javascript

  return (discKey && MAPPA_DATI_INDEX[discKey]) || null;
}
```
- Input: discKey (kebab-case)
- Output: mappa dati corrispondente o `null`
- Fallback: null (il chiamante gestisce il caso `!dati`)

# CML-411 — Live Pages Visual Baseline Gate

## Header

- Tipo: docs-only
- Baseline precedente: CML-410
- Runtime: non modificato

## Decisione

Prima di ulteriori interventi runtime occorre acquisire una baseline visuale reale della pagina pubblicata.

I mock visuali non devono essere considerati stato implementato.

Ogni analisi successiva deve distinguere tra:

1. proposta grafica;
2. codice reale;
3. vista pubblicata osservata in browser.

## Viste minime da osservare

Home, Curricolo, Revisione, Processo, Finale in verifica, Fonti, Sezioni generali, Progetta, Programmazione annuale, UDA modello, Esportazioni, Guida, menu mobile.

## Verdict

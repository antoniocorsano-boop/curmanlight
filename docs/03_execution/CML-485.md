# CML-485 — Cloudflare React root routing

## Obiettivo

Correggere la mappatura del deployment Cloudflare affinché l'applicazione React aggiornata sia pubblicata alla radice del dominio e il runtime legacy resti disponibile sotto `/legacy/`.

## Modifica

- React build pubblicata in `_site/` con `VITE_BASE_PATH=/`;
- snapshot legacy copiato in `_site/legacy/`;
- aggiunti controlli espliciti su `_site/index.html` e `_site/legacy/index.html`;
- nessuna modifica al runtime applicativo o ai dati curricolari.

## Contratto di pubblicazione

- `/` → applicazione React corrente;
- `/legacy/` → runtime legacy monolitico;
- asset pubblicati esclusivamente da `_site`.

## Stato

`CML_485_CLOUDFLARE_REACT_ROOT_ROUTING_READY_FOR_REVIEW`

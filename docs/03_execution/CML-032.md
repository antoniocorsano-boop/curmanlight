## Architettura

### Dati

Contenuti curricolari definiti in `DATA.curricoloIstituto`:

```
curricoloIstituto: {
  "2012": { name, stato, descrizione, fonti,
    ordini: { infanzia: { discipline: [ { nome, traguardi, obiettivi } ] },
              primaria: { ... },
              secondaria: { ... } } }
  "2025": { name, stato, descrizione, fonti,
    ordini: { infanzia: { ... }, primaria: { ... }, secondaria: { ... } } }
}
```

### UI

- **Tabbar**: pulsante "Curricolo di istituto" 📘 (desktop) / `mbb-cur` (mobile)
- **Container**: `#tab-curricolo` (nascosto finché selezionato)
- **Version switcher**: `curricolo-version-tabs` — "Quadro 2012" / "Quadro 2025"
- **Stato documento**: `curricolo-state` con messaggio specifico per versione
- **Indice navigabile**: `curricolo-index` con pill per disciplina + scroll fluido
- **Contenuto**: per ordine (Infanzia/Primaria/Secondaria) > disciplina > traguardi > obiettivi
- **Fonti**: `curricolo-ref` con link MIM
- **Note**: `curricolo-note` — "Il presente curricolo di istituto è un documento di lavoro. Non sostituisce la delibera del Collegio Docenti e degli organi competenti."

### Integrazione

- `setTab('curricolo')` con sincronizzazione bottom bar + sidebar nascosta
- `renderCurricoloIstituto(version)` chiamata su selezione tab e cambio versione
- CSS inline per version tabs, stato, indice, fonti, note
- Mobile bottom bar: pulsante `mbb-cur` (🏫Curr.)
- Mobile menu overlay: voce "Curricolo di istituto" e "Esporta"

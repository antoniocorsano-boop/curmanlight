## Patch plan operativo

### STEP 1 — Leggere blocco Home reale

Cercare in `index.html`:

```text
currentTab==="home"
home-discipline-status
home-inline-banner
home-discipline-open-btn
Consulta il curricolo
setTab('curricolo')
```

Identificare il blocco Home già esistente. Non sostituire tutta la Home se esistono già card/sezioni funzionanti.

### STEP 2 — Inserire chip contesto

Nel blocco introduttivo della Home, vicino al titolo o al banner principale, aggiungere un elemento:

```html
<div class="home-context-chip" id="home-context-chip" aria-label="Contesto di lavoro"></div>
```

Se esiste già un riepilogo contesto/profilo, riusarlo invece di crearne uno duplicato.

### STEP 3 — Render chip contesto

In una funzione che viene richiamata quando Home viene renderizzata o aggiornata, aggiungere:

```js
function updateHomeContextChip(){
  var el=document.getElementById("home-context-chip");
  if(!el) return;
  el.textContent=getWorkContextChip();
}
```

Poi richiamare `updateHomeContextChip()` in almeno uno dei punti già esistenti:

```text
renderProfileSummary()
renderSidebar()
updateHomeDisciplinePanel()
refreshCurrentDisciplineViews()
setTab('home') / render Home
```

Scegliere il punto meno invasivo. Preferenza:

```js
updateHomeDisciplinePanel()
```

perché è già una funzione Home-specific e viene richiamata da `renderSidebar()`.

### STEP 4 — Allineare microcopy Home

Se la Home contiene già accessi operativi, allinearli a:

```text
Cosa vuoi fare oggi?
Consulta il curricolo
Prepara una progettazione
Esporta un documento
Verifica fonti e limiti
```

Non introdurre nuove sezioni se bastano piccole modifiche testuali.

### STEP 5 — Azioni ammesse

Usare solo navigazione esistente:

```js
setTab('curricolo')
setTab('didattica')
setTab('esportazioni')
setTab('guida')
```

Se uno dei tab ha nome diverso nel runtime, usare quello reale già esistente.

Non creare nuove funzioni di routing.

### STEP 6 — Validazione umana compatta

Inserire o mantenere una frase breve:

```text
La validazione resta umana. L'app prepara materiali di lavoro, non approva il curricolo.
```

Deve essere visibile in Home, ma non dominante.

### STEP 7 — CSS minimo

Se serve una classe nuova, mantenerla minima:

```css
.home-context-chip{
  display:inline-flex;
  align-items:center;
  gap:6px;
  font-size:12px;
  line-height:1.35;
}
```

Preferire classi/stili già esistenti se disponibili.

### STEP 8 — Mobile

Verificare che:

```text
chip non esca dal viewport;
card azioni restino verticali;
nessun pannello laterale venga introdotto;
la frase di validazione non occupi troppo spazio.
```

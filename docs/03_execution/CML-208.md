## Four UX Changes Implemented

### 1. Helper Text for Multi-File Selection (P1)

Added clarifying text near both file pickers:
- Department: `"Puoi selezionare uno o più file .cml contemporaneamente."`
- Referent: `"Puoi selezionare uno o più file .cml contemporaneamente."`

Rendered as `<span class="department-import-helper">` and `<span class="referent-validation-helper">` below the local-only notice.

### 2. Mixed-Discipline Warning with Discipline List (P1)

Enhanced the mixed-discipline chip to list actual disciplines:
- Before: `"ℹ️ Sono presenti discipline diverse: il dato è segnalato ma non blocca la lettura."`
- After: `"ℹ️ Sono presenti discipline diverse: Tecnologia, Italiano. Il dato è segnalato ma non blocca la lettura."`

Uses `summary.disciplines.join(", ")` from existing `buildDepartmentImportSummary` and `buildReferentValidationSummary`.

### 3. Duplicate Chip in Referent File List (P1)

Referent file list now renders duplicate warnings as prominent chips:
- `<span class="department-check-chip" style="background:#fff3e0;color:#e65100">Duplicato probabile</span>`
- Warnings rendered as blue chips
- Missing fields rendered as red chips

This aligns referent visual treatment with department behavior.

### 4. Compact Import Summary Banner (P2)

Added a compact banner after batch import results:
- Department: `"Importati N file: M validi, K non riconoscibili, X proposte totali, Y discipline."`
- Referent: `"Importati N file: M validi, K non riconoscibili, X esiti totali, Y discipline."`

Rendered as green-background banner (`background:#e8f5e9`) above the proposal/outcome list.

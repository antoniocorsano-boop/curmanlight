# CML-483 — Percorso guidato di valutazione per docenti non tecnici

## Stato

`REMOTE_BRANCH_INITIALIZED_IMPLEMENTATION_PENDING`

## Obiettivo

Introdurre nel runtime reale di CurManLight un percorso guidato, locale e reversibile, destinato a docenti non tecnici, che consenta di:

- utilizzare alcune funzioni significative dello strumento;
- riflettere sull'esperienza d'uso;
- annotare osservazioni facoltative;
- esportare localmente una restituzione leggibile;
- non trasmettere dati all'esterno.

## Branch

`codex/cml-483-guided-teacher-evaluation`

## Vincoli operativi

- nessuna modifica diretta a `main`;
- nessun merge o pubblicazione;
- nessun backend, autenticazione o telemetria;
- nessun dato identificativo obbligatorio;
- nessuna alterazione dei dati curricolari;
- sincronizzazione obbligatoria tra `index.html` e `_published_snapshot/netlify-current/index.html`.

## Percorso previsto

1. Orientarsi nella Home.
2. Consultare una disciplina.
3. Provare un'attività non distruttiva.
4. Valutare guida e supporti contestuali.
5. Restituire punti di forza, difficoltà e suggerimento prioritario.

## Requisiti minimi

- accesso dalla Home tramite comando provvisorio;
- note facoltative per ciascuna tappa;
- possibilità di saltare, tornare indietro, interrompere e riprendere;
- salvataggio esclusivamente locale;
- esportazione locale del risultato;
- cancellazione esplicita dei dati salvati;
- comportamento non bloccante rispetto alla navigazione ordinaria.

## Controlli attesi

- assenza di errori console;
- verifica desktop e mobile;
- apertura, sospensione, ripresa e chiusura;
- esportazione corretta;
- assenza di richieste di rete riferibili al percorso;
- nessuna modifica a proposte, revisioni o contenuti curricolari.

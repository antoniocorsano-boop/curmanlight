# CML-494 — Consultation Search and Result Orientation

## Obiettivo

Rendere la consultazione React più rapida da esplorare e più leggibile quando sono attivi più criteri.

## Modifiche

- aggiunta ricerca testuale nei contenuti curricolari;
- ricerca normalizzata, non sensibile a maiuscole e accenti;
- ricerca estesa all’intera unità curricolare strutturata;
- aggiunto riepilogo del numero di risultati;
- aggiunti indicatori visibili per ordine, nucleo e termine cercato;
- aggiunto comando per cancellare solo la ricerca;
- aggiunto ripristino completo dal relativo stato vuoto;
- azzeramento filtri disabilitato quando non serve;
- aggiornato il contratto automatico B02.

## Vincoli rispettati

- dati curricolari invariati;
- consultazione sempre read-only;
- nessuna azione decisionale;
- nessun backend o servizio remoto;
- persistenza e revisione invariati.

## Validazione richiesta

- React CI;
- CML-470 Persistence Audit;
- CML-473 Comparative Audit;
- CML-491 Mobile Visual Audit;
- preview Cloudflare.

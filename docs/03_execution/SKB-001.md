## Decisione architetturale

SchoolKB e' ammissibile solo come estensione parallela, opzionale e disattivata di default.

Condizioni vincolanti:

1. Vanilla JavaScript ES6 modules.
2. Nessun framework.
3. Nessun backend nella prima fase.
4. Nessuna dipendenza esterna oltre alle librerie ufficiali Google:
   - `https://accounts.google.com/gsi/client`
   - `https://apis.google.com/js/api.js`
5. Scope massimo consentito: `https://www.googleapis.com/auth/drive.file`.
6. Nessun accesso all'intero Drive.
7. Nessun token in `localStorage`.
8. Nessun client ID reale o credenziale nel repository.
9. Feature flag obbligatorio per eventuali slice runtime future.

# CML-507 — Navigation and roadmap realignment

## Obiettivo

Rendere visibili nella navigazione React i ruoli e i passaggi già implementati, senza creare nuove viste fittizie.

## Modifica runtime

- `Curricolo` diventa `Percorso curricolare`;
- `Didattica` diventa `Progettazione didattica`;
- la voce proposta mostra il contesto `Docente`;
- la voce processo mostra `Dipartimento · Referente`;
- la voce esportazioni chiarisce `Scambio locale .cml`;
- desktop e mobile mantengono le stesse destinazioni e lo stesso comportamento.

## Stato prodotto aggiornato

- flusso curricolare Docente → Dipartimento → Referente: sostanzialmente completo;
- progettazione annuale e UDA: ancora da implementare;
- avanzamento stimato CurManLight React 1.0 dopo CML-506: 89%;
- il 93% resta vincolato alla chiusura della progettazione didattica.

## Confini

Nessun nuovo backend, autenticazione, telemetria o percorso non funzionante. Le voci disabilitate restano non mostrate.

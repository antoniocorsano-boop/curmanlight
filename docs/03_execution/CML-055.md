### Sezioni del contratto

| Sezione | Contenuto |
|---|---|
| A | Principio generale — codice operativo locale, non password, non login |
| B | Ruoli — docente, consultatore, dipartimento/coordinatore, referente |
| C | Azioni protette — `departmentOutcomeExport`, `referentOutcomeImport`, `referentReportGeneration` |
| D | Azioni libere — viewer, proposta docente, sezioni generali, guide, esempi, validazione |
| E | Comportamento UX — overlay modale, testi, pulsanti, errore, blocca di nuovo |
| F | Codici operativi — `CML2025` predefinito, cambio manuale nel codice |
| G | Persistenza — `sessionStorage`, no `localStorage` |
| H | Sicurezza e limiti — aggirabile, non protezione dati, disclosure |
| I | Compatibilità — nessuna modifica a schema `.cml`, file esportati, flusso docente |
| J | Smoke test — 13 test per implementazione futura |
| Appendice | Codice JavaScript di riferimento minimo |

### Azioni protette confermate

| Azione | Costante | Blocco |
|---|---|---|
| Esportazione esito dipartimentale `.cml` | `departmentOutcomeExport` | 🔒 |
| Import esiti dipartimentali lato referente | `referentOutcomeImport` | 🔒 |
| Generazione report gruppo curricolo | `referentReportGeneration` | 🔒 |

### Azioni libere confermate

Viewer curricolo, sezioni generali, guide, esempi, proposta docente `.cml` (export e import), personalizzazione proposta, validazione/lettura esiti senza generazione report.

### Persistenza scelta

| Aspetto | Scelta |
|---|---|
| Storage | `sessionStorage`, chiave `roleAccessGranted` |
| `localStorage` | ❌ Non usare |
| Durata | Chiusura scheda o blocco manuale |
| Sblocco multiprocedura | Sbloccando una azione, tutte e tre sono sbloccate |

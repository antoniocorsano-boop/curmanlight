## Passo 5 — Analisi del rendering effettivo

### `renderMappaDisciplinare()`
- **Input reale**: variabile globale `*_MAPPA_DATI` selezionata via `mappaDisciplinaCorrente`
- **Campi obbligatori**: `disciplina` (stringa)
- **Campi facoltativi**: `struttureSostanziali[]`, `nodiDisciplinari[]`, `progressioneVerticale[]`, `decisioniCurricolari[]`
- **Fallback**: se `mappaDisciplinaCorrente` non ha una `*_MAPPA_DATI` corrispondente, mostra messaggio "vuoto"
- **Discipline supportate**: solo tecnologia, matematica, italiano — le altre 4 mostrano sezione vuota

### `renderDidattica()` (pannello strutturato TECNOLOGIA_NORM)
- **Input reale**: `TECNOLOGIA_NORM_DATA.unitaApprendimento`
- **Campi letti**: id, ordine, classe, fascia, ambito, competenza, traguardo, obiettivi, conoscenze, abilita, evidenze, criteriValutazione, fonte, stato
- **Filtro**: per ordine (all/infanzia/primaria/secondaria)
- **Fallback**: se `TECNOLOGIA_NORM` non definito, lascia contenuto vuoto
- **Discipline supportate**: solo Tecnologia

### `DATA[selDisc]` (Curriculum editing)
- **Input reale**: `DATA[selDisc]` dove `selDisc` in ("Tecnologia", "Italiano", "Matematica", "Scienze", etc.)
- **Struttura attesa**: `{traguardi: [...], obiettivi: [...]}` — formato editing, non JSON
- **Mostra**: traguardi + obiettivi con stato di modifica/revisione

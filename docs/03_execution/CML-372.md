## Mandatory Slice Header

## Obiettivo
Introdurre `getUnitsForDiscipline(discKey)` come punto unico di accesso alle unità di apprendimento, eliminando i 4 branch speciali per Tecnologia e uniformando l'accesso ai dati per tutte le 14 discipline.

### Aggiunte

**1. `getUnitsForDiscipline(discKey)`** (dopo `MAPPA_DATI_INDEX`, prima di `getDisciplineEvidenceData`)
```javascript
function getUnitsForDiscipline(discKey) {
  return (discKey && ALL_CURRICULUM_DATA[discKey]?.unitaApprendimento) || [];
}
```
- Input: discKey (kebab-case)
- Output: array di unità apprendimento o `[]`
- Fallback: `[]` (silenzioso, il chiamante itera su array vuoto)
- **Non accede mai a `TECNOLOGIA_NORM_DATA`**
- Tutte le discipline usano lo stesso percorso dati

## Rischi residui
- `TECNOLOGIA_NORM_DATA` è dead code (~3.5 KB) — non eliminato, aspetta CML-373
- `getUnitsForDiscipline` usa optional chaining (`?.`) — richiede browser moderni (IE11 no, Chrome 80+/Firefox 74+/Safari 13.1+)
- Se una disciplina non ha `unitaApprendimento` in `ALL_CURRICULUM_DATA`, `getUnitsForDiscipline` restituisce `[]` → rendering vuoto (comportamento atteso)

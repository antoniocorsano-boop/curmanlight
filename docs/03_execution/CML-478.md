# CML-478 — Temporal Applicability Data and Resolver

## Obiettivo

Implementare il primo motore locale e deterministico che risponde alla domanda:

> Quale quadro nazionale si applica a questa combinazione di anno scolastico, ordine, classe e disciplina?

La slice implementa il contratto CML-477 senza modificare i curricoli disciplinari e senza collegare ancora il risultato alla UI.

## Baseline

- `main`: `706595d5b56a42eee28de4e2d14c7ed29b50c25d`
- contratto: `docs/02_system/DUAL-CURRICULUM-TEMPORAL-APPLICABILITY-CONTRACT.md`
- precedente card Applicabilità: stato prudente `da verificare`

## Implementazione

### Dataset normativo locale

File:

```text
curman-react/src/data/temporal-applicability.json
```

Contiene:

- schema `cml-temporal-applicability-v1`;
- quadri `IN_2012` e `IN_2025`;
- avvio transizione `2026/2027`;
- piena applicazione secondaria `2028/2029`;
- piena applicazione primaria `2030/2031`;
- applicazione completa all'infanzia dal `2026/2027`;
- eccezione Storia, primaria classe terza, `2027/2028`;
- fallback prudente `da_verificare`.

### Resolver puro

File:

```text
curman-react/src/lib/temporal-applicability.ts
```

Esporta:

```ts
resolveCurriculumApplicability(context)
```

Input:

- `academicYear`;
- `order`;
- `classYear` quando richiesto;
- `discipline` quando rilevante.

Output:

- `status`;
- `framework`;
- `ruleType`;
- `message`;
- `sourceRefs`.

Tipi di regola:

- `pre_transizione`;
- `avvio_infanzia`;
- `coorte_progressiva`;
- `anticipazione_transitoria`;
- `piena_applicazione`;
- `fallback`.

## Regole implementate

### Prima del 2026/2027

Restituisce `IN_2012`.

### Infanzia

Dal 2026/2027 restituisce `IN_2025` per tutte le sezioni.

### Primaria

La coorte IN 2025 avanza di una classe per anno:

| Anno | Classi IN 2025 | Classi IN 2012 |
|---|---|---|
| 2026/2027 | 1 | 2-5 |
| 2027/2028 | 1-2 | 3-5, salvo Storia classe 3 |
| 2028/2029 | 1-3 | 4-5 |
| 2029/2030 | 1-4 | 5 |
| 2030/2031 | 1-5 | nessuna |

### Secondaria di primo grado

| Anno | Classi IN 2025 | Classi IN 2012 |
|---|---|---|
| 2026/2027 | 1 | 2-3 |
| 2027/2028 | 1-2 | 3 |
| 2028/2029 | 1-3 | nessuna |

### Eccezione Storia

Nel 2027/2028, Storia in classe terza primaria restituisce `IN_2025` con tipo `anticipazione_transitoria`, mentre le altre discipline della stessa classe restano su `IN_2012`.

## Controlli automatici

File:

```text
curman-react/tools/check-cml478-temporal-applicability.mjs
```

Verifica:

- versione schema;
- date di avvio e piena applicazione;
- presenza dell'eccezione Storia;
- 16 casi rappresentativi dal 2025/2026 al 2030/2031;
- fallback per anno scolastico non valido;
- presenza del contratto pubblico nel resolver.

Comando:

```text
npm run test:cml478
```

Il comando è inserito in `.github/workflows/react-ci.yml` prima della build. La build `tsc -b` verifica la compilazione del resolver TypeScript.

## Limiti intenzionali

CML-478 non introduce:

- collegamento alla Home o alla card Applicabilità;
- selezione automatica dei dati curricolari 2012/2025;
- duplicazione o modifica dei curricoli disciplinari;
- regole istituzionali configurabili;
- stato di approvazione del curricolo d'istituto;
- modifica degli export;
- backend, autenticazione o telemetria;
- deploy.

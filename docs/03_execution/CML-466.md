# CML-466 — B03 WorkDecision Local Persistence

## Scopo

Introdurre una persistenza locale minima e versionata per le scelte B03, mantenendo il ponte legacy necessario ai consumatori non ancora migrati.

## Implementazione

- nuovo payload `cml-work-decisions-v1`;
- chiave locale dedicata `curmanlight:work-decisions:v1`;
- salvataggio coordinato di `workDecisioni` e proiezione legacy `decisioni`;
- caricamento sicuro con validazione strutturale minima;
- migrazione prudente da payload `cml-local-v3` limitata alle decisioni legacy;
- cancellazione esplicita del solo payload B03;
- salvataggio automatico dopo registrazione o riapertura di una scelta.

## Esclusioni

- nessun database Dexie;
- nessuna sincronizzazione remota;
- nessun trasferimento tra dispositivi;
- nessuna migrazione dell'intero stato applicativo;
- nessuna rimozione del modello legacy.

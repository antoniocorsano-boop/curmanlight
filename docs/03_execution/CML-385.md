# CML-385 — PM-03 Navigazione mobile

## Mandatory Slice Header
- Macroprogramma: PM-03 Orientamento
- Backlog: UX-020 (Navigazione mobile poco esplicita)
- Dipende da: CML-384 (Home de-densificazione)
- Tipo slice: runtime micro-intervento
- Perimetro: migliorare orientamento su dispositivi mobili

## Obiettivo

Migliorare l'orientamento dell'utente su dispositivi mobili, rendendo più chiara la navigazione tra Home, Curricolo, Esportazioni e Guida. Principio guida: **dove sono, dove posso andare, cosa posso fare**.

## Interventi eseguiti

### 1. Etichette bottom bar mobile
**Prima:**
- Home
- Curricolo
- Didattica
- Esporta
- ☰ Menu

**Dopo:**
- Home
- Curricolo
- Compila
- Esportazioni
- ☰ Menu

**Motivo:** 
- "Didattica" → "Compila": più orientato all'azione, coerente con il compito principale
- "Esporta" → "Esportazioni": coerenza con tab desktop, nome completo più chiaro

### 2. Indicatore sezione corrente
**Prima:**
- Bordo superiore 2px
- Background attivo: #f5f6ff (molto chiaro)

**Dopo:**
- Bordo superiore 3px
- Background attivo: #e8eaf6 (più visibile)
- Font-weight: 800 (grassetto extra)

**Motivo:** Rendere immediatamente riconoscibile la sezione corrente anche con visione periferica.

### 3. Icona Menu
**Prima:** ☰<br>Menu (testo puro)

**Dopo:** ☰<br><span>Menu</span> (testo in span per coerenza strutturale)

**Motivo:** Mantenere coerenza con gli altri bottoni che usano span per le etichette.

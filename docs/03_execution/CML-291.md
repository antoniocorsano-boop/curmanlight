## Problema UX-009

**Badge/etichette stato con significato non sempre univoco tra sezioni. Interpretazioni divergenti su stato e affidabilita contenuti.**

Stato pre-slice: PARZIALMENTE RISOLTO (CML-283, CML-285) — badge normalizzati in Curriculum. Il residuo riguardava:

1. **Visuale**: lo stesso `badge--readonly` appariva con stili diversi in sezioni diverse:
   - Curriculum (`tecnologia-norm-badge`): box indaco con bordo, `border-radius:10px`
   - tab-generali (`gen-badge`): pillola indaco senza bordo, `border-radius:999px`
   - Guida (inline styles): grigio, `border-radius:4px`

2. **Testo**: "Sola consultazione" (Curriculum/Guida) vs "Consultazione — non modificabile" (tab-generali) per lo stesso stato semantico

3. **Guida note**: badge con stili inline che bypassavano il Design System

## Interventi applicati

### CSS

1. **`.gen-badge` allineato a `.tecnologia-norm-badge`** (riga CSS ~519):
   - `border-radius:999px` → `border-radius:10px`
   - `padding:3px 9px` → `padding:3px 8px` (allineato)
   - aggiunto `border:1px solid #9fa8da`

2. **Nuova regola `.guida-note .badge--readonly, .guida-note .badge--prototype`** (dopo riga ~1063):
   - fornisce display/padding/border-radius/font-size/font-weight alla Guida senza inline styles
   - lascia che i colori del Design System (`background`, `color`, `border`) si applichino correttamente

### HTML

3. **tab-generali** (riga ~2027): "Consultazione — non modificabile" → "Sola consultazione"

4. **Guida note** (riga ~2279): rimossi gli stili inline da entrambi i badge (`badge--prototype` e `badge--readonly`). Ora usano solo classi Design System + nuova regola CSS.

### Non modificato

- `tecnologia-norm-badge` in Curriculum (gia normalizzato in CML-283/285) — invariato
- `badge--warning` e `badge--prototype` in Curriculum — invariati
- `evidence-badge` in Didattica — invariato (specifico del contesto evidenze)
- `esport-role-badge` in Esportazioni — invariato (indica ruoli, non stato)
- Nessuna modifica a contenuti disciplinari, schema `.cml`, export/import, storage, service worker

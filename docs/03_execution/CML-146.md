## Tool creato

**File:** `tools/generate-static-mappa-dati.mjs`

### Funzionalità

1. Riceve come argomento il percorso del JSON disciplinare
2. Usa la pipeline esistente:
   - `json-to-mappa-dati-adapter.mjs` → MAPPA_DATI ricco
   - `to-runtime-mappa-dati-transformer.mjs` → shape runtime flat
3. Genera su stdout un blocco JS statico con `const NOME_MAPPA_DATI = { ... };`
4. Non scrive file su disco
5. Non modifica runtime
6. Non usa dipendenze esterne
7. Produce errori chiari su stderr
8. Esce con codice non-zero se la pipeline fallisce

### Uso

```bash
node tools/generate-static-mappa-dati.mjs content/curriculum/tecnologia.normalized.json
```

### Output

Il tool stampa su stdout un blocco JavaScript valido con:

- `const TECNOLOGIA_MAPPA_DATI = { ... };`
- `disciplina: "Tecnologia"`
- `struttureSostanziali: [...]` (6 items)
- `nodiDisciplinari: [...]` (6 items)
- `progressioneVerticale: [...]` (10 items)
- `decisioniCurricolari: [...]` (2 items)

## Dettaglio fix

### GAP 1 ‚ÄĒ `_published_snapshot/netlify-current/index.html:2570`

**Prima:**
```js
role:"department",
disciplineWorkStatus:"completed",
```

**Dopo:**
```js
role:"department",
discipline: disciplines[0] || "",
disciplineWorkStatus:"completed",
```

`disciplines[0]` usa il primo elemento dell'array `disciplines` gia calcolato dalla funzione (disciplina unica se il dipartimento importa una sola materia, o la prima se ne importa piu di una). Se non ci sono discipline, usa stringa vuota.

### GAP 2 ‚ÄĒ `_published_snapshot/netlify-current/index.html:2545`

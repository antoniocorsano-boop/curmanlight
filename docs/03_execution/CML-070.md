# CML-070 — UDA Read-Only Prototype

**Stato:** ✅ Completed  
**Data:** 2026-06-23  
**Tipo:** Runtime + docs

## Obiettivo

Implementare il primo prototipo UDA read-only nell'area Didattica, usando una o due unità normalizzate di Tecnologia come base, senza editor, senza salvataggio, senza export nuovo e senza modificare lo schema `.cml`.

## Rapporto con CML-069

CML-069 ha selezionato l'Opzione A (UDA read-only da unità Tecnologia) e definito il perimetro di CML-070: 1–2 schede UDA modello, 19 campi, layout espandibile, microcopy prudente. Questo CML esegue quella decisione.

## Scelta read-only

Confermata. Nessun campo editabile, nessun salvataggio, nessun export. L'utente consulta esempi di UDA modello, non compila documenti.

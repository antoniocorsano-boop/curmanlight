## Causa tecnica della regressione

Le funzioni setTab e setMappaDisciplina erano definite nello script inline ma non erano esposte nell�oggetto window, quindi non erano raggiungibili dagli attributi onclick presenti nel markup HTML.

## Fix applicato

Sono state aggiunte le seguenti assegnazioni alla fine dello script inline, subito prima del tag </script> chiuso:

`javascript
window.setTab = setTab;
window.setMappaDisciplina = setMappaDisciplina;
`

Questo espone le funzioni gi� esistenti nello scope globale, rendendole disponibili agli handler inline.

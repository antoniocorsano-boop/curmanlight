# CurManLight — Scheda per il Referente Curricolo

## Compiti del referente

Il referente curricolo coordina il gruppo di lavoro che elabora il curricolo verticale d'istituto. Con CurManLight, il referente:

- raccoglie gli esiti di tutti i dipartimenti;
- importa i file `.cml` ricevuti;
- genera un report riassuntivo del lavoro svolto;
- utilizza il report per la discussione nel gruppo di lavoro.

## Passo 1 — Raccogliere gli esiti dipartimentali

Ogni coordinatore di dipartimento invia al referente il file `.cml` con l'esito del proprio dipartimento. Conservare tutti i file in una cartella sul computer.

## Passo 2 — Importare gli esiti in CurManLight

1. Aprire [https://curmanlight.netlify.app](https://curmanlight.netlify.app) nel browser.
2. Nella scheda "Revisione", scorrere fino alla sezione **"Verifica referente curricolo"**.
3. Usare il pulsante **"Importa esiti dipartimentali .cml"** per caricare ciascun file `.cml` ricevuto.
4. Verificare che tutti gli esiti siano visualizzati correttamente nel riepilogo.

Nella schermata di riepilogo è possibile vedere, per ciascuna proposta, l'esito assegnato dal dipartimento. Le voci contrassegnate con l'indicatore ✏️ sono proposte personalizzate (modificate dal docente).

## Passo 3 — Generare il report del gruppo di lavoro

1. Usare il pulsante **"Scarica report gruppo di lavoro"**.
2. CurManLight produce un file Markdown con: sintesi generale, quadro per disciplina, punti da chiarire, elementi senza esito, evidenze e traccia per la discussione.
3. Scaricare il file sul computer.

## Passo 4 — Usare il report in riunione

Il report è un documento di lavoro che riassume:

- il numero totale di proposte ricevute;
- il numero di proposte per dipartimento;
- il dettaglio di ciascuna proposta con il relativo esito;
- l'elenco delle proposte "da chiarire" che necessitano di approfondimento.

Il report va discusso nel gruppo di lavoro curricolo, che valuta la completezza del lavoro, decide sugli eventuali chiarimenti richiesti e prepara la proposta finale per gli organi competenti.

## Controllo dei punti da chiarire

Prestare particolare attenzione alle proposte contrassegnate come "da chiarire". Il gruppo di lavoro deve:

- verificare se il chiarimento è stato fornito;
- decidere se la proposta può essere approvata, riformulata o accantonata;
- annotare la decisione finale.

## Precisazione importante

- Il report generato da CurManLight **non è un atto deliberativo**.
- Il curricolo aggiornato richiede la validazione del gruppo di lavoro e la delibera degli organi collegiali competenti (Collegio Docenti, Consiglio d'Istituto).
- CurManLight è uno strumento di supporto alla gestione del materiale di lavoro, non sostituisce la discussione e la decisione umana.

# CurManLight — Guida rapida utente

## Cos'è

CurManLight è una web app che aiuta il tuo istituto a gestire l'aggiornamento del curricolo verticale. Puoi consultare il curricolo nelle versioni IN 2012 (vigente) e IN 2025 (bozza), fare proposte di modifica, discuterle in dipartimento e generare report di sintesi.

Funziona su computer, tablet e smartphone. Puoi anche installarla come app sul dispositivo; dopo la prima visita funziona anche senza connessione internet.

## URL

[https://antoniocorsano-boop.github.io/curmanlight](https://antoniocorsano-boop.github.io/curmanlight)

Non serve installare nulla: apri il browser e collega l'indirizzo.

## Sezioni principali

| Sezione | Cosa trovi |
|---|---|
| **Home** | Dashboard con accesso rapido a tutte le aree. Da qui puoi scegliere cosa fare |
| **Curriculum** | Il curricolo della tua scuola in due versioni: IN 2012 (testo vigente) e IN 2025 (bozza delle nuove Indicazioni Nazionali) |
| **Fonti** | Premessa, profilo dello studente, valutazione, inclusione, continuità, orientamento |
| **Definitivo** | Versione finale del curricolo, quando sarà approvata |
| **Competenze e progettazione** | Evidenze per competenza, programmazione annuale, bozze di UnitÃ  di Apprendimento (UDA) |
| **Revisione** | Area riservata per la proposta docente, la validazione dipartimentale e la verifica del referente |

## Accesso alle funzioni riservate

Alcune funzioni (fare proposte, validare in dipartimento, verificare come referente) sono protette da un codice ruolo. Se non hai ancora il codice, chiedilo al referente curricolo.

## Come usare l'app

### Consultare il curricolo

1. Apri l'app.
2. Clicca su **Curriculum**.
3. Scegli la versione: **IN 2012** (vigente) o **IN 2025** (bozza).
4. Usa l'indice a sinistra per navigare tra le discipline.
5. In **Fonti** trovi le sezioni generali.

### Fare una proposta (docente)

1. Apri l'app e configura il tuo profilo (disciplina, ordine di scuola).
2. Vai alla scheda **Revisione**.
3. Modifica i contenuti che vuoi proporre.
4. Clicca **"Scarica proposta .cml"**.
5. Invia il file al coordinatore del tuo dipartimento.

### Validare in dipartimento (coordinatore)

1. Raccogli le proposte dei docenti (file `.cml`).
2. In **Revisione**, sezione **"Validazione dipartimentale"**, importa i file.
3. Per ogni proposta, scegli l'esito: confluita, riformulata, assorbita o da chiarire.
4. Clicca **"Esporta esito dipartimento .cml"**.
5. Invia il file al referente curricolo.

### Verificare come referente

1. Raccogli gli esiti dei dipartimenti (file `.cml`).
2. In **Revisione**, sezione **"Verifica referente curricolo"**, importa i file.
3. Clicca **"Scarica report gruppo di lavoro"**.
4. Usa il report per la discussione nel gruppo di lavoro.

## Esempi dimostrativi

Nella cartella `esempi_cml/` trovi file di esempio per provare il flusso senza usare dati reali.

## Accessibilità

L'app è stata verificata con strumenti automatici e risulta utilizzabile con lo screen reader. Per miglioramenti o difficoltà, segnala al referente curricolo.

## Cosa ricordare

- **CurManLight è uno strumento di lavoro**, non un sistema di approvazione.
- I file `.cml` restano sul tuo computer. Nessun dato viene inviato a server esterni.
- Non inserire dati personali (nomi, cognomi, dati di alunni) nei file di prova.
- Le decisioni ufficiali sul curricolo restano al Collegio Docenti e al Dirigente Scolastico.

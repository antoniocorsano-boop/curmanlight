Oggetto: Invito alla prova controllata di CurManLight

Gentile [DS / referente / coordinatore / docente],

ti invito a partecipare alla prova controllata di CurManLight, l'applicazione web che supporta la revisione del curricolo verticale d'istituto.

**Cos'è CurManLight**
CurManLight è una PWA (Progressive Web App) che permette di consultare il Curriculum nelle versioni IN 2012 (vigente) e IN 2025 (bozza), esplorare le Fonti (sezioni generali), accedere all'area Competenze e progettazione (evidenze, programmazione annuale, bozze UDA) e sperimentare il flusso di proposta, validazione e sintesi tramite file `.cml`. È disponibile all'indirizzo:

[https://antoniocorsano-boop.github.io/curmanlight](https://antoniocorsano-boop.github.io/curmanlight)

L'app può essere installata sul dispositivo e funziona anche offline dopo la prima visita.

**Cosa ti chiediamo**
Ti chiediamo di dedicare circa 30–45 minuti a esplorare l'app e le funzioni indicate nel foglio di prova che riceverai. Dopo la prova, ti chiediamo di condividere le tue osservazioni compilando la scheda allegata o rispondendo liberamente.

**Accesso alle funzioni riservate**
Alcune funzioni (proposta docente, validazione dipartimentale, verifica referente) sono protette da un codice ruolo. Se non hai ancora ricevuto il codice, richiedilo al referente curricolo.

**Durata della prova**
1–2 settimane.

**Cosa è importante sapere**
- CurManLight è materiale di lavoro. Non sostituisce le delibere del Collegio Docenti o del Dirigente Scolastico.
- I dati restano sul tuo computer. Nessun dato personale viene raccolto dall'app.
- Non inserire dati personali o sensibili nei file di prova.

**Cosa faremo con le osservazioni**
Le tue osservazioni ci aiuteranno a migliorare lo strumento prima di un uso più esteso. Non verranno usate per approvare automaticamente contenuti curricolari.

Resto a disposizione per chiarimenti.

Grazie per la collaborazione.

# CML-001 — Viste guidate e sezioni generali in sola lettura

## Verdetto

`CML_001_VISTE_GUIDATE_READONLY_READY`

## Obiettivo

Realizzare una prima versione statica di CurManLight con navigazione guidata, sezioni generali consultabili e bloccate, area di revisione disciplinare separata e funzioni di esportazione locale.

## Viste implementate

1. Home
2. Documento generale
3. Curricolo attuale
4. Indicazioni 2012
5. Nuove Indicazioni 2025
6. Revisione disciplinare
7. Fonti ed esportazione

## Garanzie funzionali

- Le sezioni generali sono rese in card senza campi di input.
- La vista Documento generale espone badge “Sola lettura”.
- Le modifiche sono confinate alla vista Revisione disciplinare.
- Le proposte sono salvate solo nel browser tramite localStorage.
- L’esportazione è locale: copia testo, scarica JSON, stampa browser.

## Vincoli rispettati

- Nessun backend.
- Nessuna autenticazione.
- Nessuna API esterna.
- Nessun modulo Netlify.
- Nessuna dipendenza esterna.
- Nessuna modifica automatica al documento ufficiale.

## File principali

- `index.html`
- `styles.css`
- `app.js`
- `src/data/content.js`
- `netlify.toml`

## Controlli manuali consigliati

- Aprire `index.html` in browser.
- Verificare la navigazione da desktop e mobile.
- Verificare che Documento generale non contenga input.
- Verificare che Revisione disciplinare consenta l’editing delle sole proposte.
- Verificare copia riepilogo, download JSON e stampa.

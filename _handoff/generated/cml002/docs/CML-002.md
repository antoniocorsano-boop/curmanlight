# CML-002 — Documento originale integrato

## Obiettivo

Integrare il documento originale del curricolo nello strumento statico CurManLight, senza trasformare le sezioni istituzionali in aree modificabili.

## Fonte

- `Curricolo disciplinare WORD.docx`

## Perimetro

- Sezioni generali: sola lettura.
- Discipline: consultazione strutturata per area/disciplina.
- Revisione: proposte locali, modificabili, non ufficiali.
- Esportazione: riepilogo locale.

## Esclusioni

- Nessun backend.
- Nessuna autenticazione.
- Nessun modulo Netlify.
- Nessuna API esterna.
- Nessuna riscrittura automatica del documento ufficiale.

## Verdetto

`CML_002_DOCUMENTO_ORIGINALE_INTEGRATO_READY`
